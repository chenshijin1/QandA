# QandA
### 1.环境搭建
`请在python 2.7 的环境下运行`
> cd 到工程环境下 在命令行下输入下面命令

- [x] pip install -r requirements.txt

### 2.运行服务器

> 在工程环境下 在命令行下输入下面命令

- [x] python manage.py runserver

### 3.enjoy it
